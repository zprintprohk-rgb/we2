# Togthr UI 改造 PRD（We2 → Togthr 情感共生空间）

> **目标**: 把"效率工具风格 SaaS"改成"情感共生空间"
> **对标**: Sky: Children of the Light + Headspace + Calm + Linear + Tamagotchi
> **核心调性**: 治愈系 + 科技感 + 反差萌 + 隐私感
> **目标人群**: 通用（不限年龄 / 性别 / 关系状态）

---

## 1. 视觉系统

### 1.1 配色
| 用途 | 颜色 | 说明 |
|---|---|---|
| **背景渐变** | 莫兰迪淡紫 → 暖粉 / 深海蓝 → 极光绿 | 不用纯白，不用纯黑 |
| **主品牌** | `#A78BFA` (violet-400) → `#EC4899` (pink-500) | 渐变 |
| **文字** | 深色模式优先：`#FAFAFA` (text), `#A1A1AA` (muted) | |
| **卡片背景** | `backdrop-blur(16px) + rgba(255,255,255,0.05)` | 玻璃拟态 |
| **强调色** | 金色 `#FCD34D` (隐藏款皮肤) | 限定/付费 |

### 1.2 字体
| 用途 | 字体 | 体现 |
|---|---|---|
| **标题** | Playfair Display (衬线体) | 优雅 / 成熟 |
| **正文** | Nunito (圆体无衬线) | 可爱 / 年轻 |
| **像素点缀** | Press Start 2P (像素风) | 游戏感 |

### 1.3 圆角 + 阴影
- 所有卡片：`border-radius: 24px`（鹅卵石）
- 按钮：`border-radius: 9999px`（药丸）
- 阴影：柔和 + 长投影（Tailwind: `shadow-2xl shadow-violet-500/20`）

| 原 We2 | Togthr 升级 |

| 静态 | 宠物**挖掘动画**（拿铲子刨土） |

### 5.3 Virtual Pet → 共生形态
| 原 We2 | Togthr 升级 |
|---|---|

| 标签筛选 | 装修风格 = 日记情绪（暖色 / 冷色 / 节日） |
| 个人隐私 | 筑巢过程是**两个人的共同回忆** |

## 6. 隐私社区（替代"广场"）

### 6.1 定位
**"树洞"** —— 绝对安全、绝对匿名

### 6.2 UI（参考 Reddit / Soul 深色界面）
- 头像**完全随机生成**（不显示用户真实头像）
- 昵称可每周改一次
- 没有点赞数（只有"抱抱"数 = 软性互动）
- 没有"踩" / "举报"按钮（避免网络暴力）
- 发帖后 24 小时可撤回（防冲动）

### 6.3 内容引导
- 不允许"年龄差话题"显性讨论
- 引导**情绪表达**而非身份讨论
- 例：「今天很开心」>「我是大叔我喜欢小女生」

## 9. 暗色模式（默认）

### 9.1 设计原则
- **默认深色**（不是可选）
- 浅色作为辅助（用于打印 / 截图分享）
- 深色 = **避风港**感（不是黑客帝国）

### 9.2 配色（深色）
| 元素 | 颜色 |
|---|---|
| 背景 | `#0F0F1A` (深夜紫) |
| 卡片 | `rgba(255,255,255,0.05)` + 模糊 |
| 文字 | `#FAFAFA` (主), `#A1A1AA` (次) |
| 强调 | `#A78BFA` (violet), `#EC4899` (pink) |
| 警示 | `#EF4444` (red, **不要用纯红**) |
